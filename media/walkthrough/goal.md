# Goal steering

Two levels of steering shape what Master Hand suggests:

- **Project goal** — broad intent, set with *Master Hand: Set Project Goal* (e.g. "Ship the settings UI rewrite").
- **Current focus** — the immediate task. Pin it with *Master Hand: Set Current Focus*, or leave it empty and Master Hand infers it from recent edits, changed files, and diagnostics.

Changing either goal clears stale suggestions and regenerates. Goals persist per workspace.
